懒人笔记：Sea.js 模块化开发

引入sea.js文件

新建一个index.html文件，在body最下方引入sea.js

seajs配置

将 Jquery 文件放入 lib/jquery/1.10.1 中，下面这段代码就是将 Jquery 的目录指向 lib/jquery ， alias 为设置别名，方便调用

seajs.config({
 paths: {
 'jquery': 'lib/jquery'
 },

 alias: {
 'jquery': 'jquery/1.10.1/jquery'
 }

 });

Jquery 模块化封装

define(function() {
 //Jquery 源码
 return jQuery.noConflict();
});

封装模块

define(function(require, exports, module) {

var $ = require('jquery'),

    obj = {
        aa: function() {
            $('body').css({
                background: '#000'
            });
        }
    };

module.exports = obj;

});

module.exports是模块提供给外部的接口，在上面代码中就是 obj 这个对象。

require() 是个模块加载方法，参数 ‘jquery’ 是在seajs.config 中配置过的，所以浏览器就会自动加载jquery文件

启动模块

返回index.html中，在页面底部增加启动代码

seajs.use(‘source’, function(source){
        source.aa();
    });

seajs.use() 是模块启动器，会自动加载 ‘source’ 模块， 然后 Callback 事件中就可以访问该模块。
