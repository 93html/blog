原生js实现$.each()方法

假设页面上有这些P元素&lt;p&gt;1&lt;/p&gt; &lt;p&gt;2&lt;/p&gt; &lt;p&gt;3&lt;/p&gt; &lt;p&gt;4&lt;/p&gt;

$('p').each(function(){
  console.log(this.innerHTML);
});
`</pre>

## 如何用原生的js实现上面这个each方法呢？

首先要理解这段代码的原理，这段代码可以分成3个部分，`$('p')`、`each()`和一个callback函数, 然后再扩展object对象，在扩展方法中调用`each()`。

### 首先来看`$('p')`

`$()`其实是一个函数
<pre>`function $(){
  return;
}
`</pre>
想要用这个函数返回页面所有的p元素可以这样写
<pre>`function $(tag){
  var result;
  if(typeof tag ==='string')
    result = document.getElementsByTagName('p');

    return result;
}

console.log($('p').length); // 4
`</pre>
OK，搞定！jQuery的选择器的原理也不是很难嘛～

### 再来看each()

`each()`要做的事情就是传入一个数组对象，然后枚举出它每个元素，不就是循环嘛！So easy
<pre>`function each(arr){
  if(typeof arr === 'object'){
    for (var i = 0; i &lt; arr.length; i++){
    }
  }
}
`</pre>

#### 万事具备只差callback

callback只是一个函数，再每次遍历的时候去执行这个函数，所以只许对each函数略加修改即可
<pre>`function each(arr, callback){
  if(typeof arr === 'object'){
    for (var i = 0; i &lt; arr.length; i++){
      callback();
    }
  }
}
`</pre>
OK！劳资写好了，根本没有难度！跑下看看！
<pre>`var $p = $('p');

each($p, function(){
  console.log(this.innerHTML);
});
`</pre>
此时打开控制台看到的结果：
![](http://ww2.sinaimg.cn/large/80a06c8djw1esizdsk1h6j210q08s427.jpg)

![](http://ww4.sinaimg.cn/large/80a06c8djw1esizk2j6w6j20g40g4gm8.jpg) **wtf？？**
为什么`this`是这玩意？！
好吧，这里的this属于全局性调用，也就是指向了window，[关于this的用法请看这里](http://www.ruanyifeng.com/blog/2010/04/using_this_keyword_in_javascript.html "undefined")，**那么我要如何重新指定this呢？**

### 改变this指向的神器：`Call()`

> 当调用一个函数时，可以赋值一个不同的 this 对象。this 引用当前对象，即 call 方法的第一个参数。

通过 call 方法，你可以在一个对象上借用另一个对象上的方法，比如Object.prototype.toString.call([])，就是一个Array对象借用了Object对象上的方法。
简单来说，call方法就是在调用函数的同时，指定该函数的this指向call的第一个参数，并且传递若干个参数。
OK!有办法了，还等什么！来改造一下each函数
<pre>`function each(arr, callback){
  if(typeof arr === 'object'){
    for (var i = 0; i &lt; arr.length; i++){
      callback.call(arr[i], i, arr[i]);
      }
    }
}

var $p = $('p');

each($p, function(){
  console.log(this.innerHTML);
});
`</pre>
![](http://ww3.sinaimg.cn/large/80a06c8djw1esj37438qlj2122094tah.jpg)
COOL！这就是俺要得效果～

### 最后一步

我想要把each方法直接作为对象的方法，这里要使用prototype对object对象进行扩展，[对prototype的使用请点击此处](http://blog.93html.com/javascripe-object-function-and-class-function/ "undefined")
<pre>`Object.prototype.each = function(func) {
  return each(this, func);
};
`</pre>
Allright! 我也可以写出高逼格的代码啦～
![](http://ww4.sinaimg.cn/large/80a06c8djw1esj3gtimbqj20jg0jgjsi.jpg)
<pre>`$('p').each(function() {
  console.log(this.innerHTML);
});
