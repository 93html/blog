懒人笔记：Sea.js 模块化开发

引入sea.js文件

新建一个index.html文件，在body最下方引入sea.js

seajs配置

将 Jquery 文件放入 lib/jquery/1.10.1 中，下面这段代码就是将 Jquery 的目录指向 lib/jquery ， alias 为设置别名，方便调用

seajs.config({
 paths: {
 'jquery': 'lib/jquery'
 },

 alias: {
 'jquery': 'jquery/1.10.1/jquery'
 }

 });

Jquery 模块化封装

define(function() {
 //Jquery 源码
 return jQuery.noConflict();
});

封装模块

define(function(require, exports, module) {

var $ = require('jquery'),

    obj = {
        aa: function() {
            $('body').css({
                background: '#000'
            });
        }
    };

module.exports = obj;

});

module.exports是模块提供给外部的接口，在上面代码中就是 obj 这个对象。

require() 是个模块加载方法，参数 ‘jquery’ 是在seajs.config 中配置过的，所以浏览器就会自动加载jquery文件

启动模块

返回index.html中，在页面底部增加启动代码

seajs.use(‘source’, function(source){
        source.aa();
    });

seajs.use() 是模块启动器，会自动加载 ‘source’ 模块， 然后 Callback 事件中就可以访问该模块。

论程序员如何在星巴克装逼

  1. 进门后沉着地走到服务台，不要抬头看任何价目表，显得自己经常来，对价目表很熟
  2. 一边摸钱包一边向服务员MM熟练地说：你好，给我一杯Grande的美式
  3. 拿好咖啡后找一个靠窗的座位坐下，最好是玻璃外墙，好让路人看到你的屏幕
  4. 坐下要翘起二郎腿，背躺的低一点，一定要随意，不能拘谨
  5. 翻开电脑一定要是睡眠的，一解锁就是满屏幕的代码
  6. 桌上最好有一本笔记本，翻开一页上必须有各种流程图和原型图
  7. 电脑最好是MAC，不是MAC的话切记不要插着电源，没电了偷偷充一下再拔掉
  8. 程序员怎么能用鼠标？扔掉扔掉
  9. 将IDE设置成多栏显示，让屏幕上的代码尽量多一点
  10. 开个命令行工具，时不时切换过去打敲个命令，然后让屏幕刷刷刷的跑代码
  11. 电脑在桌上放久了也可以放在腿上
  12. 千万不要开着QQ
  13. 在浏览器里打开各种英文网站，时不时的切换到浏览器中，选中一段文字，边看边点头，看一会后做出一种恍然大悟的表情
  14. 喝咖啡动作一定熟练潇洒，一只手拿起咖啡放到嘴边小口小口的喝，喝完不要发出任何声音，缓缓地放到一边，全程目光不能离开屏幕
  15. 临走时不要关闭任何窗口，直接合上电脑，夹起就走
    祝大家装逼愉快 ^_^

懒人笔记:Javascript中的对象方法和类方法 

function People () {
    //类方法
    this.say = function(){
        console.log('Class');
    }
}

People.prototype.sayHi = function() {
    //为People添加一个类方法，所有People的实例都可以访问
    console.log('Hi!');
};

People.say = function(){
    //为People添加一个对象方法，仅以People为对象时可访问
    console.log('Object');
}

var p = new People();

p.say();//Class

People.say();//Object

p.sayHi();//Hi

People.sayHi();//error

How to solve: Win7 screen brightness slider disappeared

有时候笔记本电源插拔后，想要调整屏幕亮度时却发现那个亮度调节条不见了，原因不是很清楚，可能和驱动有关，在网上搜索一圈后找到了解决办法

  1. Start > Control Panel > Hardware and Sound > Device Manager
  2. Once the Device Manger box opens expand the “Monitors” tab
  3. Left Click and Uninstall the monitor listed
  4. Then select the Scan for Hardware Changes Icon
  5. Sounds bizarre, but the adjust display brightness slider should reappear.
